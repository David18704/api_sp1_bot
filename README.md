# api_sp1_bot
api sp1 bot
### Задание:
 Создан Telegram-бот, который обращается с заданной периодичностью к API сервиса Практикум.Домашка,  опрашивает API сервиса Практикум.Домашка и проверяет статус отправленной на ревью домашней работы;
при обновлении статуса анализирует ответ API и отправляет соответствующее уведомление в Telegram;
логирует свою работу и сообщает о важных проблемах сообщением в Telegram.
